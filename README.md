  The Famous Person Network Analysis project models relationships between well-known individuals using graph-based machine learning techniques. Each person is represented as a node, and their connections (collaborations, influence, or interactions) are represented as edges.

This project demonstrates how network science + AI/ML can be used to analyze influence, detect communities, and uncover hidden patterns in social or professional networks.
Feaures:
.Graph construction using NetworkX
.Visualization of relationships between famous people
.Centrality analysis to identify influential individuals
.Community detection (cluster analysis)
.Extendable to real-world datasets (news, social media, Wikipedia)
Tech Stack:
.Python
.NetworkX
.Matplotlib
.Pandas
